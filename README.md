# Masa - Truck Tracking

## To install and run locally
MongoDB client should be installed locally.

### Server :
```js
cd server
npm install
npm run dev
```
### Websockets :
```js
cd websockets
npm install
npm start
```
### Client :
```js
cd client
npm install
npm run build
npm run dev
```

## To run with Docker:
```js
 docker-compose up
```
